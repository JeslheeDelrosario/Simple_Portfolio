Project 3 (Thesis Project)

TISA - Towards Intelligent Student Assistance

An AI-powered learning management system designed for College of Science Students at Bulacan State University. The platform integrates advanced AI tutoring capabilities with robust academic management features, including SSO, calendar management, weather integration, and task tracking.

  • 🤖 AI Tutor (TISA) – Context-aware assistant with conversation continuity
  • 🔐 SSO Authentication – Keycloak integration with JWT & role-based access
  • 📅 Calendar Management – Classroom scheduling, holiday calendar, event management
  • 🌤️ Weather Integration – Real-time weather & 7-day forecast for Bulacan
  • ✅ Task Management – Create, prioritize, and track task completion
  • 📚 Learning Management – Course enrollment, lesson viewer, progress tracking
  • 🔔 Real-time Notifications – Bell icon with unread count & history
  • 🛠️ Admin Dashboard – Room, faculty, curriculum, user management
  • ⚡ Performance – Redis caching, optimized queries, rate limiting
React 18 + TypeScript Vite TailwindCSS + Radix UI Node.js + Express Prisma ORM PostgreSQL + Redis Keycloak SSO Docker + Docker Compose OpenAI/Anthropic/Gemini OpenWeatherMap API
⚙️ System Architecture & Challenges
  • Multi-provider AI service integration (OpenAI, Anthropic, Google Gemini)
  • Redis caching layer for weather API (3-hour TTL, reduced latency)
  • Keycloak SSO with JWT authentication and role-based access control
  • Dockerized microservices (PostgreSQL, Keycloak, Redis) for consistent dev/prod parity
  • Google Calendar OAuth integration for personalized event syncing
  • Real-time notification system with unread count and history
  • Environment configuration management across dev/production
📦 Status: Production Ready 📝 License: MIT 🐳 Dockerized 🔐 SSO Enabled
